# Monorepo Summary

This repository contains a monorepo managed by [Turborepo](https://turborepo.dev/docs). It implements a web-based note-taking application called "Astronote" that is strongly influenced by a MacOS application called [The Archive](https://zettelkasten.de/the-archive/). Notable features include:

## The Omnibar

An "omnibar" at the top of the screen that allows the user to:

- Search for existing notes
- Create new notes

## The Sidebar

Below the omnibar there is a sidebar on the left-hand side of the screen. As the user searches for existing notes in the omnibar, matching entries are displayed in this sidebar.

## The Content View

The remaining area of the screen (to the right of the sidebar) is the "content view." This portion of the screen is where the user:

- Views the content of existing notes
- Creates new notes
- Edits existing notes
